# Player Alias Naming Guide

Rules for turning real player names into funny-but-instantly-recognisable aliases.
The goal: a fan reads the alias and smiles because they already know who it is.

**The one-step rule:** change the minimum — one letter, one sound, one syllable —
to create something that lands as a pun or wink without obscuring the identity.

---

## Rule 1 — Word Reveal (highest priority)

If swapping 1–2 letters exposes a real English word (or any widely-known word),
do it. The humour comes from the hidden meaning suddenly being obvious.

| Real Name | Alias | What changed | Why it's funny |
|---|---|---|---|
| Bukayo Saka | Sake | drop one letter | Japanese rice wine |
| Mohamed Salah | Salsa | a→sa at end | the dance / the dip |
| Harry Kane | Cane | K→C | walking stick |
| Declan Rice | Rice | kept as-is | already a food |
| Raheem Sterling | Stirling | e→i | real Scottish city, also sounds posh |
| Karim Benzema | Benzene | a→e | chemistry textbook cameo |
| Thomas Müller | Miller | umlaut→il | direct English translation of Müller |
| Jude Bellingham | Billingham | e→i | real industrial town in England |
| Jack Grealish | Garish | e-removed, rearranged | adjective meaning flashy / loud |
| Lamine Yamal | Yamal | first-name drop | real Arctic peninsula in Russia |

---

## Rule 2 — Minimal Respelling (PES classic)

Change one vowel or one consonant in the most prominent syllable.
The result should still be pronounceable the same way, just look slightly off.

Preferred swaps (pick the one that sounds most natural):
- Vowels: **a ↔ e**, **o ↔ u**, double a/l/r → single, or vice-versa
- Consonants: **c ↔ k**, **y ↔ i**, **ph ↔ f**, **ck → k**
- Endings: **-son → -sen**, **-er → -ar**, **-ic → -ik**, **-ez → -es**

| Real Name | Alias | What changed |
|---|---|---|
| Erling Haaland | Harland | aa→ar (real English surname) |
| Kevin De Bruyne | De Bruin | drop the 'y' (valid Dutch variant) |
| Trent Alexander-Arnold | Trent Arnold | drop the hyphen name, keep both ends |
| Bernardo Silva | Silverio | suffix swap (-a → -erio) |
| Bruno Fernandes | Fernandez | s→z (Spanish variant) |
| Luka Modric | Modrik | c→k |
| Toni Kroos | Kruse | oo→use (German surname, different person) |
| N'Golo Kanté | Kante | drop accent, clean spelling |

---

## Rule 3 — Anglicisation

For foreign names where a natural English equivalent exists, use the English version.
Works best when the English version has a funny or mundane feel next to the player's fame.

| Real Name | Alias | Note |
|---|---|---|
| Thomas Müller | Miller | German for "miller" — he's literally "the Miller" |
| Robert Lewandowski | Lewandowsky | -ski → -sky, anglicised Polish |
| Florian Wirtz | Wirts | drop the z, now sounds like an English surname |
| Marcus Rashford | Rashford | kept — already sounds like a made-up English village |
| Phil Foden | Foden | kept — short, already odd-sounding |

---

## Rule 4 — Suffix Flavour

Add or swap a cultural suffix to give the name character.
Use sparingly — only when the base name is too plain to alter.

| Suffix | Feel | Example |
|---|---|---|
| -inho | Brazilian flair | Fodeninho, Salahinho |
| -ski / -sky | Eastern European | Sancho → Sanchoski |
| -son | Scandinavian | Saka → Sakason |
| -ito | Spanish diminutive | Pedro → Pedrito (already used in football!) |

---

## Rule 5 — Mononym Transform (stars known by one name)

For players recognised by a single name, apply Rules 1–4 to **that name** rather than
keeping it verbatim. The one name is the base for the transformation, not the final output.

| Real Name | Alias | Rule applied | Note |
|---|---|---|---|
| Kylian Mbappé | Killian | minimal respelling (y→i) | common Irish name, sounds identical |
| Vinicius Jr. | Vinicious | word reveal (add -ous) | becomes an adjective — fitting |
| Pedri González | Petri | minimal respelling (d→t) | petri dish — lab experiment of a player |
| Neymar Jr. | Neyman | minimal respelling (r→n) | sounds like "neigh man" |
| Zlatan Ibrahimovic | Sultan | anagram / word reveal | Z→S, rearrange: Zlatan ≈ Sultan — he'd approve |

---

## Priority Order

1. **Word Reveal** — if a pun is there, use it
2. **Minimal Respelling** — one-letter tweak, same pronunciation
3. **Anglicisation** — natural English equivalent exists
4. **Suffix Flavour** — when nothing else works
5. **Mononym Transform** — for stars known by one name, transform that name

When in doubt: say the alias out loud. If you'd recognise the player in 0.5 seconds, it works.
